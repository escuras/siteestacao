Projeto desenvolvido para a disciplina de sistemas distribuídos
do mestrado em Computação móvel na Escola Superior de Tecnologia
e Gestão do Instituto Politécnico da Guarda.
Esta página funciona com o recurso a uma API externa, onde são
obtidos recursos como criação e edição de utilizadores, configurações
e registo de temperatura (a ideia original pretendia enriquecer este registo
com mais sensores, com vista a criação de uma autêntica estação metereológica).
Qualquer utilizador com registo pode descarregar a aplicação existente no
menu configurações, ainda poderão encontrar um manual para configuração da
aplicação e ligação do raspberry a um sensor de temperatura (com mais tempo mais
sensores irão sendo acrescentados).
